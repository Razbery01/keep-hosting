
Then edit `package.json` to add these scripts (merge with existing `dev`, `build`, `lint`, `preview` — do not delete them):

```json
"scripts": {
  "dev": "vite",
  "build": "tsc -b && vite build",
  "lint": "eslint .",
  "preview": "vite preview",
  "test": "vitest",
  "test:watch": "vitest --watch",
  "test:ci": "vitest run --reporter=dot"
}
```

Rationale:
- `vitest` (without --run) is the dev default; developers running `npm run test` get watch mode only if they pass `--watch`. The 01-VALIDATION.md sampling commands use `npm run test -- --run` which overrides watch.
- `test:ci` uses `run` + `dot` reporter so it exits and is quiet for CI/verification sampling.
- Do NOT use `--watch` in any default script — the Nyquist validator will flag it.

After installing, verify vitest is on the version listed in package.json:
```bash
npx vitest --version
```
