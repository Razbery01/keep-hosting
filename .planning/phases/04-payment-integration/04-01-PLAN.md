    - verifyPayFastSignature returns false when params are tampered

  <action>
Create `supabase/functions/_shared/payfast.ts` with:

1. `import md5 from 'npm:md5'` — NOT crypto.subtle (which does not support MD5).

2. `generatePayFastSignature(params: Record<string, string>, passphrase: string): string`
   - Filter out key `'signature'` and keys with empty/undefined values
   - Sort remaining keys alphabetically
   - Map to `key=encodeURIComponent(value).replace(/%20/g, '+')` pairs
   - If passphrase is non-empty, append `passphrase=encodeURIComponent(passphrase).replace(/%20/g, '+')`
   - Join with `&`, compute `md5()`, return lowercase

3. `verifyPayFastSignature(params: Record<string, string>, receivedSignature: string, passphrase: string): boolean`
   - Calls `generatePayFastSignature(params, passphrase)` and compares lowercase to `receivedSignature.toLowerCase()`

4. Export constants:
   - `PAYFAST_SANDBOX_URL = 'https://sandbox.payfast.co.za/eng/process'`
   - `PAYFAST_LIVE_URL = 'https://www.payfast.co.za/eng/process'`
   - `getPayFastUrl(sandbox: boolean): string` — returns sandbox or live URL

5. `generateApiSignature(headerParams: Record<string, string>, passphrase: string): string`
   — Same algorithm as generatePayFastSignature, used for PayFast API calls (cancel subscription).

NOTE: This file uses `npm:md5` Deno import. It will NOT be importable in Vitest (Node). The pay03 tests will use source-scan assertions on this file (reading the .ts source and asserting patterns), consistent with the established deploy test pattern.
